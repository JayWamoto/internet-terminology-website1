Domain Name System

Definition

DNS translates domain names into IP addresses.

How it Works

When a user enters a URL, DNS finds the corresponding IP address of the server.

Key Features

  • Fast lookup system
  • Hierarchical structure
  • Essential for Internet navigation

Example

example.com → 142.250.x.x
