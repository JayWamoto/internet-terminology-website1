World Wide Web

Definition

The World Wide Web (WWW) is a system of interlinked hypertext documents accessed over the Internet using web browsers.

How it Works

Users request web pages using a browser. The request is sent to a web server, which responds by sending HTML files that are displayed on the screen.

Key Features

  • Uses hyperlinks to connect pages
  • Runs on HTTP/HTTPS protocols
  • Accessible globally

Example

Visiting a website like Google or YouTube.
