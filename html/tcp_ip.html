TCP/IP

Definition

TCP/IP is a communication protocol suite used to connect devices on the Internet.

How it Works

TCP breaks data into packets and ensures delivery, while IP handles addressing and routing.

Key Features

  • Reliable data transmission
  • Packet-based communication
  • Universal Internet standard

Example

Sending an email or loading a webpage.
