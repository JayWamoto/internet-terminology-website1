Web Server

Definition

A web server is a system that stores and delivers web content to users.

How it Works

It receives requests from browsers and responds with requested files such as HTML pages.

Key Features

  • Handles multiple requests
  • Stores website files
  • Supports HTTP protocol

Example

Apache, Nginx servers.
