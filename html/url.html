Uniform Resource Locator

Definition

A URL is the address used to access resources on the Internet.

How it Works

It specifies the protocol, domain name, and path to locate a resource.

Key Features

  • Unique identifier
  • Includes protocol (http/https)
  • Human-readable format

Example

https://www.example.com
